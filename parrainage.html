Programme de parrainage

Partagez RESPEKTUS® avec vos proches et soyez récompensées ensemble. Chaque parrainage vous offre des avantages exclusifs.

Comment ça marche ?

1

Trouvez votre code

Votre code de parrainage unique se trouve dans votre profil, section "Programme de fidélité". Il est généré automatiquement à la création de votre compte.

2

Partagez-le

Envoyez votre code à vos amies, votre famille ou partagez-le sur les réseaux sociaux. Votre filleule le saisit lors de son inscription sur l'application.

3

Profitez des récompenses

Dès que votre filleule crée son compte avec votre code, vous recevez toutes les deux vos récompenses automatiquement.

Questions fréquentes

Combien de personnes puis-je parrainer ?

Il n'y a pas de limite ! Plus vous parrainez, plus vous cumulez de récompenses.

Quand est-ce que je reçois ma récompense ?

Votre réduction de 5€ et vos 50 points sont crédités automatiquement dès que votre filleule crée son compte avec votre code.

Ma filleule doit-elle acheter quelque chose ?

Non, la récompense est activée dès la création du compte. Aucun achat minimum n'est requis.

Où trouver mon code de parrainage ?

Dans l'application RESPEKTUS, allez dans votre Profil, puis dans "Programme de fidélité". Votre code unique y est affiché avec un bouton pour le copier et le partager.

La réduction de 5€ a-t-elle une date d'expiration ?

La réduction est valable 6 mois à compter de son attribution.

Prête à parrainer ?

Téléchargez l'application RESPEKTUS et retrouvez votre code de parrainage dans votre profil.
