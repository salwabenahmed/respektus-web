Programme de parrainage

Partagez RESPEKTUS® avec vos proches. Vous gagnez des points à chaque inscription, et plus vous parrainez, plus vous débloquez de récompenses.

Comment ça fonctionne

1

Récupérez votre code parrain

Votre code unique se trouve dans votre profil, section Programme de fidélité. Format : RSP-XXXXXX.

2

Partagez-le

Envoyez votre code à vos proches ou partagez-le sur les réseaux sociaux. La personne parrainée le saisit lors de son inscription.

3

Vous gagnez tous les deux

Dès la création du compte, vous recevez +100 points (+ bonus de paliers à partir de 2 filleuls). La personne parrainée reçoit +100 points de bienvenue. Aucun achat requis.

Questions fréquentes

Combien de personnes puis-je parrainer ?

Il n'y a pas de limite. Plus vous parrainez, plus vous cumulez de points et débloquez de paliers bonus.

Quand est-ce que je reçois mes points de parrainage ?

Les 100 points sont crédités automatiquement dès que la personne parrainée crée son compte avec votre code. Les bonus de paliers (2e, 5e, 10e, 25e filleul) s'ajoutent à ces points.

Où trouver mon code parrain ?

Dans l'application RESPEKTUS®, ouvrez votre Profil puis la section Programme de fidélité. Votre code est affiché avec un bouton de partage.

Mes points expirent-ils ?

Oui. Vos points sont remis à zéro chaque année à la date anniversaire de votre inscription. Pensez à les convertir en bons Aroma-Zone avant cette date pour ne pas les perdre. Les bons déjà générés restent valables 12 mois.

Comment recevoir mon bon d'achat Aroma-Zone ?

Une fois le palier atteint, ouvrez la section Programme de fidélité de votre profil et cliquez sur le bon souhaité. Un email contenant le code vous parvient sous 48 heures à l'adresse de votre compte.

C'est quoi le statut Ambassadeur ?

À partir de 5 000 points cumulés ou 25 personnes parrainées, vous devenez Ambassadeur RESPEKTUS®. Vous accédez aux tests produits en avant-première et bénéficiez de récompenses exclusives.

Y a-t-il une limite de points par jour ?

Certaines actions sont plafonnées pour éviter les abus : 10 points par jour pour les posts dans le Cercle (soit 2 posts maximum récompensés), 5 points par jour pour les commentaires (soit 5 commentaires maximum récompensés). Le bonus de connexion 7 jours d'affilée est limité à 1 fois par mois. Les autres actions ne sont pas limitées.
